# Doxygen Documentation

[Doxygen](https://www.doxygen.nl/) with [Doxygen Documentation Generator (VS Code Extension)](https://marketplace.visualstudio.com/items?itemName=cschlosser.doxdocgen) is used for generating documentation.

To generate documentation, run the following from the project root folder: `doxygen doxygen.config`

To view documentation generated by Doxygen, simply open the `docs/doxygen/html/index.html`-file with any web browser.
